# Lawn Care Scheduling App

This app helps you schedule and manage lawn care tasks effectively.

## Features
- Schedule lawn mowing
- Reminders for watering plants
- Track lawn care history

## Installation
1. Clone the repository
2. Install dependencies using `npm install`
3. Run the app using `npm start`
